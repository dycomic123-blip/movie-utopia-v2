## ⚠️ 常见问题

### Q: 视频不播放？
```
检查:
1. 网络连接是否正常
2. 控制台有无错误 (F12 → Console)
3. 视频文件路径是否正确
```

### Q: 布局没有变化？
```
检查:
1. 是否正确调整了窗口大小
2. 刷新页面试试 (F5)
3. 清除缓存 (Ctrl+Shift+Delete)
```

### Q: 开发服务器没运行？
```bash
# 运行这个命令启动
npm run dev
```
